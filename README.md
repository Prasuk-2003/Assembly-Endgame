# 🎮 Assembly Endgame

An interactive word-guessing game built with **React** and **Vite**, where players must uncover a hidden programming-related word before running out of attempts. Every incorrect guess eliminates an assembly language, making each game progressively more challenging.

```
👉 [https://assembly-endgame-neon.vercel.app/](https://assembly-endgame-neon.vercel.app/)
```

---

## 📖 Overview

Assembly Endgame is a browser-based word guessing game inspired by classic word puzzle games. Players reveal the hidden word one letter at a time while trying to avoid exhausting all available attempts.

The project was built to strengthen React fundamentals, including component-based architecture, state management with hooks, conditional rendering, and responsive UI design.

---

## ✨ Features

* 🎯 Random hidden word each game
* ⌨️ Interactive on-screen keyboard
* ✅ Correct and incorrect guess tracking
* 💀 Assembly language elimination after wrong guesses
* 🎉 Win and loss detection
* 🔄 Restart game functionality
* 📱 Responsive interface
* ⚡ Fast development with Vite

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Prasuk-2003/Assembly-Endgame.git
```

Navigate to the project:

```bash
cd Assembly-Endgame
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```

---

## 🎮 How to Play

1. Start a new game.
2. Click letters on the virtual keyboard.
3. Correct guesses reveal letters in the hidden word.
4. Incorrect guesses eliminate one assembly language.
5. Guess the complete word before all attempts are exhausted.
6. Click **New Game** to play again.
